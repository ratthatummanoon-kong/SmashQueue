## Topic: Badminton Matchmaking & Analytics Platform

Overview This project is a management system designed for badminton social groups (Guan). It streamlines the queueing and matchmaking process for organizers ("Hua Guan") while providing data-driven insights for players ("Luk Guan").

# Key Features

- For Organizers: Automated matchmaking system to balance teams and manage court queues efficiently.

- For Players: 
    - Personal Dashboard: View individual performance stats and skill levels (Hand P/C/B).

    - Analytics: Track win rates and win/loss records.

    - Match History: See past opponents and partner history to analyze playstyles.
